## Implementation Notes

The probe command is a thin wrapper around the existing `e2eprobe.ProbeServers()` function. The justfile cleanup removes the bash recipes that are now implemented in Go. The `just check-stale-refs` recipe should grep for old `task <cmd>` patterns to catch any missed reference updates. This task can be parallel with 3.1 and 3.2 since it doesn't depend on `pkg/e2e/`.
