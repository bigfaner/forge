# 2.3: New list-types command + TaskTypeRegistry

## Description

Create a new `forge task list-types` command that outputs all supported task types. Introduce `TaskTypeRegistry` as the centralized source of truth for task types, replacing the scattered definitions across `pkg/task/infer.go` and `pkg/prompt/data/`.

## Implementation Notes

The `TaskTypeRegistry` centralizes types currently scattered across multiple files. The `list-types` command iterates the registry and prints each entry. Consider having `infer.go` reference the registry instead of maintaining its own type list, to avoid drift. The 11 types are: implementation, fix, gate, doc-generation-summary, doc-generation-consolidate, test-pipeline-gen-cases, test-pipeline-eval-cases, test-pipeline-gen-scripts, test-pipeline-run, test-pipeline-graduate, test-pipeline-verify-regression.
