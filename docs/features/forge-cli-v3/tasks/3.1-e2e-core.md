# 3.1: pkg/e2e/ core: profile resolution + ExecRunner

## Description

Create the `pkg/e2e/` package with profile resolution logic and the injectable `ExecRunner` interface. This is the shared infrastructure that all e2e subcommands will delegate to.

## User Stories

- Story 5: `ResolveProfile` reads config.yaml and returns profile name; no profile → "no e2e profile configured" error; unknown profile → "unknown profile: <value>" error

## Implementation Notes

The `ExecRunner` interface enables unit testing without calling external tools. The `stubExec` struct uses a `map[string]execResponse` pattern for predetermined outputs. Production code sets `runner = RealExec{}` by default; tests override with `stubExec`. The package uses only Go's standard `testing` package.
