# 2.2: quick-tasks surface-key/type 填充

## Description

## Reference Files

`design/tech-design.md#Phase-Component-Map` — Phase 2 quick-tasks 变更

- When 任务 frontmatter 生成完成
- Then frontmatter 包含 `surface-key: admin-panel` 和 `surface-type: web`
- And 无 surfaces 配置的项目不填充 surface-key/type
