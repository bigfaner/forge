# 1.3: Extract magic values to constants + export isBusinessTask

## Description

Extract 10 runtime-affecting hardcoded values to `pkg/constants/forge.go`. Export `isBusinessTask` as `IsBusinessTask` to `pkg/task/` (deduplicate 3 copies). Zero behavior change.

The 10 magic values include: lock timeout, fix-task cap (3), advisory lock retry interval, and other hardcoded numbers. Extract to named constants with clear documentation. For `IsBusinessTask`, the 3 duplicates are in `cmd/validate_index.go`, `pkg/prompt/prompt.go`, and `pkg/task/add.go`.
