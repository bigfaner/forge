# Generate e2e Test Scripts

## Description

Call `/gen-test-scripts` skill to generate executable TypeScript e2e test scripts from test cases.

Generated scripts use:
- CLI tests: `child_process.execSync`
- Test framework: `node:test` + `node:assert`

## Reference Files

- `testing/test-cases.md` — Test case document (generated by T-test-1)
- `tests/e2e/config.yaml` — Test environment config

## Acceptance Criteria

- [ ] `tests/e2e/justfile-standard-vocabulary/` contains at least one spec file
- [ ] Each test() includes traceability comment `// Traceability: TC-NNN → {PRD Source}`

## User Stories

No direct user story mapping. This is a standard test generation task.

## Implementation Notes

1. Run `/gen-test-scripts` skill
2. Verify spec files exist under `tests/e2e/justfile-standard-vocabulary/`
3. After generating spec files, run `just e2e-verify --feature justfile-standard-vocabulary`; if exit 1 (unresolved `// VERIFY:` markers), resolve markers before proceeding to T-test-3
