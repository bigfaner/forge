# 1: Clean up config.go dead code and infer.go missing cases

## Description

Remove dead code and fill missing type handling in Go source files. Specifically:

1. **config.go** (`forge-cli/pkg/forgeconfig/config.go` ~line 115): Coverage defaults reference `"coding.clean"` type which was renamed to `"code-quality.simplify"`. This entry never matches any task — remove it.

2. **infer.go** (`forge-cli/pkg/task/infer.go`): `InferType` function handles auto-generated task IDs (e.g., `T-test-run`, `T-quick-*`) but lacks cases for `T-eval-journey` and `T-eval-contract`, causing them to return an empty string. Add these two cases.
