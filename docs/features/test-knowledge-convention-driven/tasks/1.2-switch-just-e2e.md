# 1.2: Switch pkg/just and pkg/e2e to forgeconfig

## Description
Update `pkg/just/just.go` and `pkg/e2e/e2e.go` to use `forgeconfig` instead of `profile`. These are leaf packages with simple profile dependencies.

**pkg/just**: Replace `profile.ReadConfig` (for ProjectType) with recipe-existence check via `just --list`. If justfile has a scoped recipe variant, pass scope; otherwise suppress.
