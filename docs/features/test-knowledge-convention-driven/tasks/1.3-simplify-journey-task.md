# 1.3: Simplify pkg/journey and pkg/task

## Description
Remove profile type dependencies from `pkg/journey/` and `pkg/task/`. These packages share types (Language, AutoConfig) so they must be updated together.

**pkg/journey**: Remove `FrameworkInfo` from `TestGenerationOpts`, remove `GenerateGoTest`/`GeneratePythonTest`/`GenerateJSTest`/`GenerateDispatched`, remove `FeatureTag`/`RegressionTag`. Keep `TestOutputDir`, `SmokeTestName`, `GeneratedTest` type.

**pkg/task**: Remove `Language` type usage, `StrategyResolver`, `TestInterfaces`, `Languages` fields. Simplify `BuildIndexOpts` from 8 fields to 5 (remove Languages, TestInterfaces, ResolveStrategy). Change AutoConfig type from `profile.AutoConfig` to `forgeconfig.AutoConfig`. Update `GetBreakdownTestTasks`/`GetQuickTestTasks` to use auto.E2eTest toggle only.

## Reference Files
- `forge-cli/pkg/journey/journey.go` — remove Generate*, tags, FrameworkInfo
- `forge-cli/pkg/task/testgen.go` — remove Language type, simplify task generation
- `forge-cli/pkg/task/build.go` — simplify BuildIndexOpts
- `docs/features/test-knowledge-convention-driven/design/tech-design.md` — I-2, I-3

## Acceptance Criteria
- `pkg/journey/journey.go` has zero `profile` imports
- `GenerateGoTest`, `GeneratePythonTest`, `GenerateJSTest`, `GenerateDispatched`, `FeatureTag`, `RegressionTag` functions removed
- `TestGenerationOpts.FrameworkInfo` field removed; `Facts string` field added
- `pkg/task/testgen.go` and `build.go` have zero `profile` imports
- `BuildIndexOpts` has 5 fields: FeatureSlug, ProjectRoot, TasksDir, IndexPath, AutoConfig (forgeconfig.AutoConfig)
- `TestTaskDef` simplified: ID, Key, Title, TestType (no Language field)
- `GetBreakdownTestTasks`/`GetQuickTestTasks` take `forgeconfig.AutoConfig` parameter
- `go build ./...` succeeds
- `go test ./pkg/journey/... ./pkg/task/...` passes (tests updated)

## User Stories
- Story 1 (non-default framework): task generation no longer language-dependent
- Story 4 (backward compatibility): auto.E2eTest toggle unchanged

## Hard Rules
- Task content no longer embeds `forge test get generate` output
- E2e task files reference Convention path: "Read docs/conventions/testing-*.md"
