# 1.4: Simplify CLI commands

## Description
Update all CLI commands in `internal/cmd/` to use `forgeconfig` instead of `profile`. Remove language/framework collection from interactive flows.

**config.go**: Simplify `forge config init` to only collect `auto.*` + `worktree`. Remove project-type/languages/interfaces collection.

**init.go**: Simplify `forge init` interactive config to match config.go simplification.

**index.go**: Remove profile calls from `forge task index`. Language resolution → removed. Interface resolution → removed. Strategy resolver → removed. Use `forgeconfig.AutoConfig` for e2e task toggle.

**add.go**: Remove profile calls from `forge task add`. Simplify index rebuild.

**test.go**: Remove subcommands: `detect`, `get generate`, `get run`, `get justfile`, `get template`, `interfaces`, `framework`. Keep: `verify`, `promote`, `run-journey`.
