# 4.1: Create record-decision skill

## Description
新建 `plugins/zcode/skills/record-decision/SKILL.md`，实现独立的 `/zcode:record-decision` 命令，通过 4 轮 AskUserQuestion 交互完成单条决策的归档，引用 `references/shared/decision-logging.md` 中的 record-decision 流程。

## Acceptance Criteria
- [ ] `plugins/zcode/skills/record-decision/SKILL.md` 文件存在
- [ ] SKILL.md 中 name 字段为 `record-decision`
- [ ] SKILL.md 定义 4 轮交互：类型（1-8）、决策描述、决策理由、关联 feature
- [ ] SKILL.md 引用 `plugins/zcode/references/shared/decision-logging.md` 中的 record-decision 步骤
