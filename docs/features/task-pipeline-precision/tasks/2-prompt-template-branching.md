# 2: Add complexity branching and search strategy to prompt templates

## Description

Modify the 4 coding prompt templates (enhancement, feature, refactor, cleanup) to support complexity-based branching: wrap Step 1.5 in conditional markers and add search strategy guidance before the implementation step.

`coding.fix` is explicitly excluded — it has a fixed 5-step flow and is auto-generated by the dispatcher.
