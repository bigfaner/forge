## Conventions Established

1. At external boundaries (YAML frontmatter, CLI string flags), use string(types.Xxx) conversion rather than changing the external interface to typed constants
2. When upstream returns []string but downstream expects typed constants, use types.SurfaceType(val) conversion at the boundary
3. Non-canonical values (e.g., 'failed' status in tree.go) should be preserved alongside typed constants for backward compatibility
4. Use types.AllStatuses()/AllPriorities() for building validation maps rather than hardcoding values
