# 3.1: Migrate pkg/forgeconfig/ SurfaceType constants

## Description

将 `pkg/forgeconfig/` 中所有 Surface Type 魔法值替换为 `types.SurfaceType` typed constants，升级 map key 类型、函数签名和推断返回值。

## Affected Files

- `forge-cli/pkg/forgeconfig/detect_surface.go`
- `forge-cli/pkg/forgeconfig/detect.go`
- `forge-cli/pkg/forgeconfig/execution_order.go`

## Reference Files

- `forge-cli/pkg/forgeconfig/detect_surface.go`: `surfacePriority` map: `map[string]int` → `map[types.SurfaceType]int`; 所有 signal maps (packageJSONSignals, goModSignals, etc.) value 改为 types.SurfaceType; inference return values "api"/"cli"/"web" → types.SurfaceXxx; inline comparisons `== "mobile"` → `== types.SurfaceMobile` (source: design/tech-design.md#Model-3)
- `forge-cli/pkg/forgeconfig/detect.go`: `KnownSurfaceTypes`: `map[string]bool` → `map[types.SurfaceType]bool` 使用 types.AllSurfaceTypes(); `SurfaceTypes()` return type upgrade (source: design/tech-design.md#Interface-2)
- `forge-cli/pkg/forgeconfig/execution_order.go`: `defaultExecutionOrder`: `[]string{...}` → `[]types.SurfaceType{...}` (source: design/tech-design.md#Model-3)

## Acceptance Criteria

- [ ] `detect.go` 中 `KnownSurfaceTypes` 类型为 `map[types.SurfaceType]bool`，或直接使用 `AllSurfaceTypes()` 生成
- [ ] `detect_surface.go` 中 `surfacePriority` map key 类型为 `types.SurfaceType`
- [ ] `detect_surface.go` 中所有 signal map values 使用 `types.SurfaceXxx` 常量
- [ ] `detect_surface.go` 中所有推断函数返回值使用 `types.SurfaceXxx`
- [ ] `detect_surface.go` 中所有 `== "web"` 等内联比较改为 `== types.SurfaceWeb`
- [ ] `execution_order.go` 中 `defaultExecutionOrder` 类型为 `[]types.SurfaceType`
- [ ] 零 Surface Type 字符串字面量残留
- [ ] `go build ./pkg/forgeconfig/...` 零错误
- [ ] `go test ./pkg/forgeconfig/...` 全部通过

## User Stories

- US2: Centralized SurfaceType Constants — forgeconfig 是 SurfaceType 魔法值最密集的区域（~50+ 处）

## Hard Rules

- Surface Type 常量值必须与原始字符串完全一致（"web", "api", "cli", "tui", "mobile"）
- detect_surface.go 的推断逻辑行为零变更

## Implementation Notes

### Test Impact
- Affected test suite(s): `forge-cli/pkg/forgeconfig/` (detect_surface_test, surfaces_test, config_test, execution_order_test)
- Expected fixture changes: test assertions中的 "web"/"api" 等字符串 → types.SurfaceWeb 等
- Risk level: medium — ~50+ replacements in detect_surface.go but purely mechanical
