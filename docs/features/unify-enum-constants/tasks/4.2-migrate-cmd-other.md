# 4.2: Migrate internal/cmd/ other files (all enums)

## Description

替换 `internal/cmd/` 中除 `cmd/task/` 之外的文件中的 Status、Priority、SurfaceType 魔法值。

## Affected Files

- `forge-cli/internal/cmd/quality_gate.go`
- `forge-cli/internal/cmd/cleanup.go`
- `forge-cli/internal/cmd/verify_task_done.go`
- `forge-cli/internal/cmd/surfaces.go`
- `forge-cli/internal/cmd/feature/feature.go`
- `forge-cli/internal/cmd/feature/feature_complete.go`
- `forge-cli/internal/cmd/worktree/helpers.go`

## Reference Files

- `forge-cli/internal/cmd/quality_gate.go`: `needsFullLifecycle()` 签名 string → types.SurfaceType; "web"/"api"/"mobile" 比较 → types.SurfaceXxx; Status 比较 → types.StatusXxx; fix task Priority "P0" → types.PriorityP0 (source: design/tech-design.md#Migration-Plan-Phase-5)
- `forge-cli/internal/cmd/surfaces.go`: `KnownSurfaceTypes = forgeconfig.KnownSurfaceTypes` 重导出——当 map key 从 string 变为 types.SurfaceType 时需同步更新 (source: design/tech-design.md#Architecture-Component-Diagram)
- `forge-cli/internal/cmd/feature/feature.go`: 内联 StatusEnum 切片 → []types.Status; Status 赋值/比较 (source: design/tech-design.md#Migration-Plan-Phase-5)
- `forge-cli/internal/cmd/worktree/helpers.go`: Status 比较 "completed" → types.StatusCompleted (source: design/tech-design.md#Migration-Plan-Phase-5)
