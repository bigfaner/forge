# 1.1: Create pkg/types/ package with typed constants

## Description

新建 `pkg/types/` 叶包，定义 Status、SurfaceType、Priority 三种 typed constants 及其辅助函数。这是整个迁移的基础——后续所有任务依赖此包。

## Affected Files

- `forge-cli/pkg/types/status.go` (NEW)
- `forge-cli/pkg/types/surface.go` (NEW)
- `forge-cli/pkg/types/priority.go` (NEW)

## Reference Files

- `forge-cli/pkg/types/status.go`: 定义 `type Status string` + 7 常量 + `AllStatuses()` + `IsTerminalStatus()` (source: design/tech-design.md#Interface-1)
- `forge-cli/pkg/types/surface.go`: 定义 `type SurfaceType string` + 5 常量 + `AllSurfaceTypes()` (source: design/tech-design.md#Interface-2)
- `forge-cli/pkg/types/priority.go`: 定义 `type Priority string` + 3 常量 + `AllPriorities()` (source: design/tech-design.md#Interface-3)

## Acceptance Criteria

- [ ] `pkg/types/` 包存在，不导入任何 forge-cli 内部包（纯标准库）
- [ ] `status.go` 定义 `type Status string` 和 7 个常量（pending, in_progress, completed, blocked, suspended, skipped, rejected），值与 `pkg/feature/constants.go` 中完全一致
- [ ] `status.go` 提供 `AllStatuses() []Status` 返回全部 7 个常量
- [ ] `status.go` 提供 `IsTerminalStatus(s Status) bool`，对 completed、skipped、rejected 返回 true
- [ ] `surface.go` 定义 `type SurfaceType string` 和 5 个常量（web, api, cli, tui, mobile）
- [ ] `surface.go` 提供 `AllSurfaceTypes() []SurfaceType`
- [ ] `priority.go` 定义 `type Priority string` 和 3 个常量（P0, P1, P2）
- [ ] `priority.go` 提供 `AllPriorities() []Priority`
- [ ] `go build ./pkg/types/...` 零错误
- [ ] `go test ./pkg/types/...` 全部通过（编写常量值正确性 + JSON 序列化兼容性测试）

## User Stories

- US1: Type-Safe Status Constants — typed constant 定义
- US2: Centralized SurfaceType Constants — SurfaceType 定义
- US3: Reliable Enum Refactoring — pkg/types/ 作为叶包

## Hard Rules

- `pkg/types/` 零内部依赖（不导入 forge-cli 任何包）
- 常量值必须与现有字符串完全一致（行为零变更）
